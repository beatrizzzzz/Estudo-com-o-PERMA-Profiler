Sobre o projeto

Este repositório reúne os códigos, bancos de dados e notebooks utilizados em um estudo acadêmico sobre bem-estar psicológico, florescimento humano e concepções de felicidade, tendo como principal referência o modelo PERMA.

A proposta do projeto é combinar diferentes formas de análise:

análise quantitativa dos escores das dimensões do PERMA;

caracterização sociodemográfica da amostra;

análise de correlação entre as dimensões do modelo;

identificação de perfis de participantes por agrupamento;

redução de dimensionalidade por PCA;

análise de respostas abertas sobre felicidade;

categorização computacional de conceitos presentes nos textos;

representação conjunta de participantes e conceitos por MCA;

produção de gráficos destinados à apresentação dos resultados em um Trabalho de Conclusão de Curso.

O repositório contém uma base em formato SPSS (.sav), uma base derivada em formato CSV e quatro notebooks Jupyter. Cada notebook corresponde a uma etapa ou conjunto específico de resultados.
